Technologies used

Microsoft Teams

Microsoft teams will be primarily used to host and record team meetings as well as to post any questions that someone may have while working on the project.

Trello

This will function as a page to store any and all information relating to the progress of our assignments and project overall, giving a full overview of what was done and when it was completed.

Github

Github will be used to host and store any code files for use on our website and acts as a place for everyone to access said files. These files can be altered by anyone in the group, so that we can work on the project collaboratively.

Google Docs

Docs will be used for storage of any large text sections that will be present on our e-commerce site and will allow for a cloud based storage section for everyone to utilize.

Google Sheets

We used Google Sheets to put the 'Estimation Times' template and work on the table collaboratively.

Invision

Invision will be our primary tool for the design of our prototypes and will be used by all groups members to create said prototypes of our MVFâ€™s.

Text Editors

As we will be creating a webpage we will need some form of advanced text editor to edit and create HTML5, CSS3, php and any other forms of files we would need for this assignment. Tools the group will be using will include common text editors such as visual studio.

Image Editor

An image editor may also be used in creating and combining some of the diagrams we need for this assignment, it doesn't really matter which editor is used as long as the file types are compatible across all team members devices.

Amazon Web Services

Our webpage will eventually need to be hosted for submission, and as such we will again be using AWS for this section of our assignment. Specifically the s3 services. We will also use AWS to create and store our databases for our online e-commerce site.

WampServer

WampServer is used to create web applications from Apache2, PHP, MySQL and MariaDB. Used to create our 'Upload Photo' and 'Superimpose Photo' features.
