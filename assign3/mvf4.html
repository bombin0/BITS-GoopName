Database - MVF 4 for OES website

Team member in charge of MVF - Arjun Sharma



Introduction:

To store information regarding product and user details we had to implement a database. We used wamp to host the database and phpMyAdmin to manage the database. This database contains 4 tables for products, users, wishlist and images.

The products table stores:

  • Product ID (Primary)
  • Product image
  • Product name
  • Product price

The Users table stores:

  • User ID (Primary)
  • Username
  • Email
  • Phone number
  • Password

The wishlist table is empty when the site is loaded and the User must log in and press the 'Add to wishlist' button on the products page to send data to the wishlist table. When the user presses the 'Add to wishlist' button, the wishlist table is populated with the details of the product that the User wants to add.

In this case the wishlist table will store:

  • Wishlist ID (Primary)
  • Product ID (Foreign)
  • Product image
  • Product name
  • Product price

The images table also is empty when the site is loaded. The user must be logged in and have uploaded an image on the products page for this table to be populated.

When the user uploads an image, the images table will store:

  • Image ID (Primary)
  • Image
  • Image text


Database setup:

To create the tables we used the following queries:

    CREATE TABLE products (
      product_id int(11) NOT NULL PRIMARY KEY AUTO_INCREMENT,
      name varchar(128) NOT NULL,
      image BLOB NOT NULL,
      price int(11) NOT NULL
    );


    CREATE TABLE users (
      idUsers int(11) AUTO_INCREMENT PRIMARY KEY NOT NULL,
      uidUsers TINYTEXT NOT NULL,
      emailUsers TINYTEXT NOT NULL,
      phoneUsers varchar(15) NOT NULL,
      pwdUsers LONGTEXT NOT NULL
    );


    CREATE TABLE wishlist (
      wishlist_id AUTO_INCREMENT INT NOT NULL,
      product_id INT NOT NULL,
      name VARCHAR(40),
      image BLOB,
      price INT,
      PRIMARY KEY (wishlist_id),
      FOREIGN KEY (product_id) REFERENCES products(product_id)
    );

    CREATE TABLE images (
      id int(11) NOT NULL AUTO_INCREMENT,
      image varchar(200) NOT NULL,
      text text NOT NULL,
      PRIMARY KEY (id)
    );
