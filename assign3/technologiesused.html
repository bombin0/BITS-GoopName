List of all tools and technologies used by the group.

Microsoft Teams

Microsoft teams is a website and an application that allows for video and audio meetings between multiple people as well as the exchange of text and files over channels. In the case of this assignment Microsoft teams was used to perform meetings with our tutor , lecturer as well as host meetings between all of our group members on a weekly basis.

Github

Github is a website and desktop application that allows users to store files within a cloud server, groups may be made from multiple users that allows said users to share files, upload files and update files all from one space. In the case of this assignment , github was primarily used for the storage of any and all html, css, php , javascript and any other files needed for the creation of our AWS hosted website.

Google docs

Google docs is basically a form of text editor such as Microsoft Word in which an individual may create a report or typed document. The difference between google docs and other text editors is that all files created using docs are stored on google's cloud servers and linked to google accounts, these files may also be worked on by multiple users at one time. Hence we primarily used google docs for collaborative work such as written documents or planning documents.

AWS

Amazon web services is a website that allows for the hosting of web pages through the upload of all the necessary code, said code is uploaded into one place, similarly to github, and the AWS service will enable the generation of a website link to which other people may view the website without having any of the files. This was used for the submission of our website.

Trello

Trello is a website dedicated to the creation and maintenance of ‘boards’ in which users may upload files, fill out information, create checklists, all in the goal of managing time and progress within a group environment. Trello allows every individual group member the ability to view the progress of other members and the group as a whole and allows the teachers to view the progress of a specific group.

Image editor

An image editor such as paint.net or photoshop was used by all group members when designing the low fidelity mockups of the MVF’s , as well as when we developed the high fidelity mockups for the EVF’s as InVsion requires a base image in order to utilize its full feature list. Again , the type of editor that was used did not matter, as long as the file type was readable by every group member.

Text Editors

Text editors such as visual studio code or notepad++ were used by all group members in order to create the code necessary for our website. It did not matter which editor each group member decided to use, as long as they saved the files in the correct format, everything was fine and worked as intended.

Wampserver

Wampserver is an application that allows a user to recreate a server environment through their local network, as such, we have used this service to test PHP files as well as any SQL databases that we needed for this assignment and to test the overall functionality of the completed website with these added functions.

Invision

InVision is a prototyping tool used to create high-fidelity mockups of whatever you need, utilizing images created by the designer and adding clickable areas and other visual features that allow a user to get a solid overview of how the final product will look. For this project we use invision to create the mock ups for our individual MVF’s as well as the EVF’s as can be seen on the ‘EVF’ section of this website.

MySQL

MySQL is a tool used to create and alter databases under the SQL language, it is relatively in depth but fairly simple to learn, especially for a smaller scale website such as the one we have created. For this assignment we used MySQL to create the database for the website which features elements from all the MVF’s such as account id’s and image names as some examples.
